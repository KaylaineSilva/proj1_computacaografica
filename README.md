## Objetivo
Desenvolver um programa que apresente uma janela envolvendo diferentes objetos, cores e transformações geométricas, conforme abaixo.

Os objetos presentes nesse trabalho foram modelados com o programa Blender pelas autoras, disponibizados na pasta Objetos. 

## Projeto desenvolvido
O projeto desenvolvido é uma cena com objetos de uma academia, contendo os seguintes objetos e transformações:
- Banco: rotação no eixo y

- Boneco: rotação e escala (objeto componente biceps)

- Halter: translação

- Garrafa

- Barra

## Comandos de teclado
- Tecla P: habilita e desabilita as malhas dos objetos

- Tecla A: rotação para a esquerda no banco

- Tecla S: rotação para direita no banco

- Tecla G: rotação para a esquerda no boneco

- Tecla H: rotação para a direita no boneco

- Seta para cima: movimento de rosca para cima (rotacionando o antebraço no eixo z com o ponto fixo e transladando o halter para cima)

- Seta para baixo: movimento de rosca para baixo (rotacionando o antebraço no eixo z com o ponto fixo e transladando o halter para baixo)

- Tecla L: aumentar o biceps

- Tecla R: reset em todos os objetos
